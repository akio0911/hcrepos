
\section{10のソフトウェア(Windows編)}
やはり，パソコンといえばWindowsでしょう(と，普通の人ぶる)．

そこで，Windowsのおすすめソフトウェアを紹介してみます．
後に紹介するものほど普通の人には不要なソフトウェアです．

\begin{enumerate}
\item Sleipnir
\footnote{http://www20.pos.to/~sleipnir/}
 and Firefox
\footnote{http://jt.mozilla.gr.jp/products/firefox/}

定番のタブブラウザです．
特別な理由がなければSleipnirだけで十分だと思いますが，
以下のような方にはFirefoxもおすすめです．

\begin{itemize}
\item Internet Explorerのレンダリングエンジンが気に食わない
\item Internet Explorer固有のセキュリティホールが気になる
\item 他人のPCを使う際に履歴などが残らないように気を使うのが面倒
\end{itemize}

なお，Sleipnirは作者がソースコードを紛失された
\footnote{http://slashdot.jp/article.pl?sid=04/11/17/1341205}
ため，開発を停止しています．


\item Noah
\footnote{http://www.kmonos.net/}

書庫の圧縮・解凍のソフトウェアです．
同じ配布元のcaldixも組み合わせて使うと必要な
DLLも一気にダウンロードできるので便利です．

ちなみに私は関連付けがいやなので，
エクスプローラのメニュー拡張からしか使いません．

\item StarSuite
\footnote{http://jp.sun.com/products/software/starsuite/}
 or OpenOffice.org
\footnote{http://ja.openoffice.org/}

オフィススイーツです．
配布物にはWriter(Word), Calc(Excel), Impress(PowerPoint)などが
含まれています(括弧内は相当するMicro\$oft製品)．

仕事で使う場合はMicro\$oft製品でないとつらいですが，
学業で使用する場合にはこれで十分です．
むしろこちらの方が \LaTeXe などとの連携性において優れています．

なお，StarSuiteはOpenOffice.orgのパッケージと考えてよいです．
素材やフォントなどが若干充実しています．
私はStarSuiteも年間ライセンスのものを購入しています．

\item xyzzy
\footnote{http://www.jsdlab.co.jp/~kamei/}

EmacsやMuleに似ているエディタです．
この原稿もxyzzyで書いています．

総合的にエディタを評価した場合，
どうしてもEmacsen
\footnote{Emacsの複数形でEmacsの類という意味}
が抜きんでているため，Windowsで気軽に使える
という観点からxyzzyを使用しています．

\item CraftLaunch
\footnote{http://hp.vector.co.jp/authors/VA012411/}

コマンドライン型のランチャです．
ソフトウェアの起動などに使います．

PuTTYと組み合わせて使うと便利
\footnote{http://taka.no32.tk/diary/?date=20040110\#p01}
です．

ホットキーとしては Ctrl + Tabを使っています．
なぜなら，Ctrl + SpaceはEmacsでのリージョンの設定に使い，
Shift + SpaceはIMEの切替に使ってしまっているからです．
なお，Windowsキーは存在しないキーボードを使っているため，問題外．

\item CLCL
\footnote{http://www.nakka.com/soft/clcl/}

クリップボードの履歴管理ツールです．
過去にクリップボードにコピーした内容をAlt + Cで簡単に復活できます．

DLLを作ることにより，機能を拡張することも可能です．
Cygwinのgcc + make でも作ることができます．
興味のある方にはMakefile差し上げます．

\item WipeOut
\footnote{http://hp.vector.co.jp/authors/VA031384/}

デスクトップを一時的に最前面にするツールです．
デスクトップにアクセスする際にウィンドウなどを動かす必要が
なくなりますので，ストレスが減ります．

\item PuTTY
\footnote{http://hp.vector.co.jp/authors/VA024651/}

telnet/sshクライアントです．
ssh2にも対応している数少ないソフトです．

*nixユーザは必携です．

\item VisualStudio 6.0 
\footnote{http://www.microsoft.com/japan/msdn/vstudio/default.aspx}
and Eclipse
\footnote{http://www.eclipse.org/}

統合開発環境です．
Windows APIを使う
C++での開発をする場合にはVisualStudio 6.0が慣れていますので，
それを使うようにしています．

Javaを使う際にはEclipseを使います．
プログラマにとってはもはや常識だと思いますので，
細かい説明はしません．

\item Cygwin
\footnote{http://www.jaist.ac.jp/~fujieda/cygwin/}

POSIXの環境を充実させるソフトウェア群です．
bash, inetd, Apache, Ruby, X.orgなどを
一気に導入することができます．

ck
\footnote{Cygwin用ターミナルエミュレータ}
もあわせて使うと便利です．


\end{enumerate}
